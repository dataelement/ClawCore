# ClawCore 🦐

> [OpenClaw](https://github.com/openclaw/openclaw) のコア版 — 魂を持つAIアシスタント。

[English](README.md) | [中文](README_CN.md) | [日本語](README_JA.md) | [한국어](README_KO.md) | [Español](README_ES.md)

ClawCoreは、OpenClawから最も核心的なパーソナリティシステムを抽出した、ミニマルで自己完結型のパーソナルAIアシスタントです。AIに「命を吹き込む」仕組みはそのまま、インフラの複雑さを取り除きました。

## 🎯 なぜClawCore？

### OpenClawとの主な違い

| 🦐 ClawCore | 🦞 OpenClaw |
|------------|------------|
| **インデックス型メモリ** — `MEMORY_INDEX.md`で目次管理、ベクトルDB不要 | ハイブリッドベクトル検索 + 埋め込みモデル |
| **軽量ハートビート** — シンプルな`setInterval`タイマー | 完全なcronシステム + サブエージェント |
| **人間とAIのフォルダ分離** — ユーザーファイルは読み取り専用 | 共有ワークスペース |
| **タスクワークベンチ** — タスクごとのフォルダ管理 | タスクフォルダの概念なし |
| **ファイル安全設計** — AIは原本を変更不可、コピーのみ処理 | より広いファイルアクセス |
| **日常PCで安全に動作** — 専用マシン不要 | サーバーデプロイ向け |

### 🔒 日常のパソコンで安全に動作

- **`user/`は読み取り専用。** AIがPDFやWordを読めても、書き込みは物理的に不可能。
- **処理は`workbench/`で行う。** ファイル編集が必要な場合、AIはまずタスクフォルダにコピー。
- **全操作に権限境界。** コードレベルで強制 — 信頼ではなくコードで保護。

## 🚀 クイックスタート

```bash
git clone https://github.com/dataelement/ClawCore.git
cd ClawCore
npm install
npm run dev
```

初回起動時：
1. LLM APIキーの設定
2. AIの「覚醒」会話で自己発見
3. デスクトップに `~/Desktop/ClawCore/` ワークスペースを作成

## ✨ 主な機能

| 機能 | 説明 |
|------|------|
| 🧬 **魂システム** | `SOUL.md`でAIが独自の個性を発展 |
| 🪪 **アイデンティティ覚醒** | 初回起動時の自己発見儀式 |
| 🧠 **インデックス型メモリ** | `MEMORY_INDEX.md`目次、オンデマンドで読み込み |
| 🔧 **スキルシステム** | AIがスキルを自主的に作成・進化、`SKILL_LOG.md`に記録 |
| 📁 **ユーザー保管庫** | ユーザーファイル読み取り専用 |
| 🛠️ **タスクワークベンチ** | タスク別フォルダ + ライフサイクル管理 |
| 💓 **ハートビートスキャン** | 定期自律スキャン、🤖プレフィックス付きタスク自動作成 |

## 🛡️ セキュリティ

- **ファイルアクセス** — シンボリックリンクを解決してからパスチェック
- **シェルコマンド** — ホワイトリスト（`ls`,`cat`等）/ ブラックリスト（`rm`,`curl`等）/ 確認メカニズム
